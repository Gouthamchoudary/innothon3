CORE INTERPRETATION RULES:
1. This is the official academic calendar for the Spring Semester of AY 2025-2026 for all undergraduate and postgraduate programs at NIT Warangal.
2. Students should interpret Saturdays and Sundays as holidays/non-class days by default unless a specific event, exam, registration task, or institute activity is explicitly written on that date.
3. Weekdays are working academic days unless marked as a holiday, exam day, registration day, or special event day.
4. The calendar contains both a date-grid view and a “Major Activities” list; both represent the same semester schedule.
5. Some dates on weekends still contain official activities, so weekend days are not universally empty.
6. One line in the source likely contains a typo: “Payment of Fee and Online Registration for Autumn Semester — July 13-19, 2025” appears inconsistent because the next item refers to AY 2026-2027 classwork starting on 20 July 2026.

CALENDAR LEGEND / EVENT TYPES PRESENT:
- Classwork Commences / Last Working Day
- Mid / End Exams
- Registration

STUDENT WEEKEND RULE:
- Sunday: holiday by default unless a specific activity is written.
- Saturday: holiday by default unless a specific activity is written.
- If a Saturday or Sunday contains registration, exams, or an institute event, that activity overrides the default holiday assumption for that date.
